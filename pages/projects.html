Logos Connection

A web platform designed to help users explore philosophy by classifying them into the philosophical school that best matches their perspectives and decisions.

Users can create an account and log in. If the account is new, the system gathers information through forms to determine the user’s philosophical school and provides resources to learn more about it.

The platform also allows users to create, edit, and delete blog posts where they can share thoughts and questions. Other users can comment on these posts, which are also classified to show how the ideas relate to the user’s assigned philosophical school.

The project is built with Flask as the framework and MySQL for storing users and blog data, combining Python for the backend with HTML, CSS, and JavaScript for the frontend.
